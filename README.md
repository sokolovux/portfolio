# portfolio

Personal portfolio website for Maxim Sokolov.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Full local stack (Vercel APIs + middleware):

```bash
npm run dev:full
```

## Build

```bash
npm run build
npm run preview
```

## Lint

```bash
npm run lint
```

## Project layout

- `src/` — React app
- `public/` — static assets
- `src/styles/design/` — Bootstrap theme and custom SCSS
- `api/` — Vercel serverless functions
- `scripts/` — prerender and crawl file generation
